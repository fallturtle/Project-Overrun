# Project Overrun

A browser-based apocalyptic survival FPS built with Three.js.

## Planned Features
- FPS movement and combat
- Dynamic weather and day/night
- Zombie AI
- Loot and inventory systems
- Vehicles
- Base building
- Radiation zones
- Survival mechanics

## Tech Stack
- HTML/CSS/JavaScript
- Three.js
- Browser-first architecture
- GitHub Pages / Cloudflare Pages compatible

## Development Strategy
This game will be built incrementally in modular systems instead of attempting all features simultaneously.

### Phase 1
- FPS movement
- Basic map
- Zombie enemies
- Shooting system
- Health system

### Phase 2
- Inventory
- Looting
- Survival stats
- Weather
- Day/night

### Phase 3
- Vehicles
- Base building
- NPC factions
- Dynamic events
